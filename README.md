Overview

This project is a Login and Registration System built using PHP and MySQL.
It provides a secure and user-friendly platform for user authentication, including registration, login, and session management.
The repository also includes a detailed project report and a ZIP archive of the full source code.

 Features

-> User registration with form validation

-> Secure login authentication

-> Password encryption using hashing

-> MySQL database integration

-> Session management and logout

-> Error handling and input validation

-> Clean and simple user interface

 Technologies Used

-> Frontend: HTML, CSS

-> Backend: PHP

-> Database: MySQL

-> Server: XAMPP
